# Чеклист проверки данных
по json легенда:
! - не добавляется в умения в листе персонажа
(заклинание) - добавляет персу заклинание которое указано в []

(?) - надо чтобы учитывался особенности КД
(?) - если предоставляется выбор между видами владения чего то то надо дать выбор(пример - автогном умение особая сборка)

Отмечайте пункты галочкой после ручной проверки корректности данных в приложении.
х - полностью проверена корректность данных
- - исправлены данные в json
+ -
